## 1. 本目錄的任務

`docs-ai/06_Development_Records/worklogs/` 存放每日的開發日誌。

## 2. 檔案命名規範

`Worklog-<YYYYMMDD>.md`

範例：
*   `Worklog-20240101.md`

## 3. Worklog 模板

```markdown
# Worklog: <YYYY-MM-DD>

## 1. 今日目標
*   完成 Ticket-XXX
*   修復 Bug-YYY

## 2. 執行紀錄
*   10:00 - 開始分析需求...
*   14:00 - 遇到 DB 連線問題，原因是...
*   16:00 - 完成實作，提交 PR #123

## 3. 遇到的問題 (Blockers)
*   (若有無法解決的問題，請記錄於此)

## 4. 明日計畫
*   繼續 Ticket-ZZZ
```
